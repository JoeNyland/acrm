About αCRM

Hello!

I'm Joe Nyland and I am the developer who has hand crafted the αCRM system for our family business: Nyland Construction. I have taken great care whilst building the system and have used the latest and greatest open-source projects and packages.

 What's so special?

Features

Simple

Given that the original use case of the application was that *any* member of the team should be able to create, manage and distribute customer related documents, no matter how tech-savvy they are, the αCRM system has been designed from the ground up to be as easy as possible to use.

Your Granny may even be able to use it, if indeed she does work your company's CRM system from time to time!

Clean

The Twitter Bootstrap framework has been utilised when building the αCRM system, which provides the simplest and cleanest user interface available to date. This allows users of the app to feel at home when using αCRM, as they are using interfaces that they already use through social media websites like Twitter and this helps to increase productivity when using the app!

Secure

From encryption to authentication, from hashing to salting and from archiving to email, your data and your client's data is handled securely in the admin system. You no longer have to worry about backups, archival and encryption when handling your client data.

Cloud Optimised

Cloud computing has revolutionised the IT industry over the past few years. Through providing flexible, low-cost and reliable services, cloud computing providers such as Amazon AWS, Microsoft Azure and Rackspace, are have bolstered the image of cloud computing to the masses. It's great to know that αCRM has been designed and built around the cloud and providing software as a service (SAAS), and has been tried and tested in these environments.

Of course, like anything though, the cloud does not come without problems or concerns. Should you prefer to run αCRM on your own infrastructure, on your own premises, you are quite welcome to do so.

Open-source

Through the αCRM system, several open-source projects have been utilised: From PHP server-side code at the core of the application, to the MySQL database which holds all of your valuable data and to the versatile jQuery JavaScript code running on the client side, a great deal of effort has been made to make the application as customisable and extensible as possible.

Because αCRM is built using community provided open-source software, it's completely free for you to use. Yes, that's correct - FREE!!! I don't ask any money from you or anything in return if you use the package, provided that it's used within the terms and conditions of the open-source license. The αCRM open-source license can be downloaded here.

Customisable

αCRM is highly customisable and flexible and customisable. Through it's open-source roots, anyone can edit it, customise it and contribute to it through the GitHub open-source collaboration platform, which is where αCRM lives!

Should you find that there is a feature that you think your business and others would benefit from and you are feeling adventurous, there's nothing stopping you from getting stuck in by "forking" the project and submitting a "Pull Request"! For the less development-orientated users, please feel free to submit a feature request in the αCRM issue manager here, or consult the help and support section of this site.
